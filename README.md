# bronto-cli

A community command-line client for the [Bronto](https://bronto.io) observability platform.

> This is a placeholder README. It will be replaced with full install,
> quickstart, and usage documentation in a follow-up task.

## Install

```sh
curl -fsSL https://raw.githubusercontent.com/svrnm/bronto-cli/main/scripts/install.sh | sh
```

Or with Go:

```sh
go install github.com/svrnm/bronto-cli/cmd/bronto@latest
```
