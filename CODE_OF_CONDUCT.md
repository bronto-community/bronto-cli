# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be constructive, assume good faith. Harassment or
discriminatory behavior is not tolerated.

Report violations privately to the maintainers via
[GitHub's report-abuse flow](https://github.com/contact/report-abuse) or by
opening a confidential security advisory on this repository.
